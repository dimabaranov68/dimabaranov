Основы МКТ

Основные положения МКТ:

Все вещества состоят из частиц(молекул, атомов).

Частицы находятся в постоянном хаотичном движении.

Частицы взаимодействуют друг с другом.

Основная формула МКТ: p=1/3m0nV^2, где p - давление, m0-масса одной частицы, n - , V - скорость.

Для примеров берётся идеальный газ(газ, частицы которого взаимодействуют между собой посредством упругого соударения(модель газа)).

Начальные формулы МКТ:

Плотность (масса на единицу объёма) ρ = m/V, где m - масса, V - объем

ρ = m/V = m0 * n = m0 / a^3, где m0 - масса одной частицы, n - концентрация

Концентрация (количество молекул на единицу объема) n = N / V, где N - количество молекул, V - объем

Количество вещества (величина, выражающая количество моль вещества равная количеству моль углерода на ту же массу) ა = N / NA = m / M, где N - количество молекул, NA - число Авогадро(6,02 * 1023 г/моль), m - масса, M - молярная масса

Молярная масса - табличные значения (находить по таблице Менделеева)

Диаметр атома равен, примерно, 10^-8 м, масса 10^-27 кг.

Вывод основной формулы МКТ:

Отсюда получим, что F0t=m0V+m0V

F0=2m0V/t

за t N частиц

F=F0N

p=F/S

N=n*v

v=V*S*t

F=2m0V^2Sn

p=2m0nV^2

В реальности молекулы двигаются хаотично, но их движение можно раскидать по трем осям, на каждой из которых можно двигаться в двух направлениях, следовательно: основная формула МКТ: p=1/3m0nV2, где p - давление, m0-масса одной частицы, n - , V - скорость.
